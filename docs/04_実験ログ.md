## v2 ベースライン（参考値）

### モデル選定: DenseNet-121 vs EfficientNet-B0

両モデルをImageNet事前学習から全層fine-tuneし、同一条件で比較。

| 指標            | DenseNet-121 | EfficientNet-B0 | 選定     |
| --------------- | ------------ | --------------- | -------- |
| パラメータ数    | 8.0M         | 5.3M            | -        |
| Val Accuracy    | 0.935        | 0.931           | DenseNet |
| ROC-AUC (macro) | **0.9908**   | 0.9871          | DenseNet |
| Macro F1        | **0.83**     | 0.80            | DenseNet |
| mel F1          | **0.62**     | 0.60            | DenseNet |
| bcc F1          | **0.92**     | 0.81            | DenseNet |

**選定理由:** 全体Accuracyは同等だが、ROC-AUC macro（0.9908 vs 0.9871）とMacro F1（0.83 vs 0.80）でDenseNetが優位。最大の差はbcc（F1: 0.92 vs 0.81）で、臨床的に重要な悪性腫瘍の検出で大幅にリード。EfficientNetはepoch 8でピーク後にval lossが増加傾向、DenseNetはLR減衰後のepoch 11-13で改善が継続し、最適化の余地を示した。

## t-SNE 埋め込み解析

Esteva et al. (2017) / Cino et al. (2025) に準拠。classifier直前の1024次元埋め込みをt-SNE (perplexity=30, max_iter=1000) で2次元に射影し、argmax判定・二値閾値判定の正誤をオーバーレイ。

### v3a t-SNE

- test N=552, argmax accuracy 0.9076, misclass 51 (9.2%), mel FN 10
- Binary (T=1.4478, threshold=0.1439): FN 7/56 (12.5%), FP 57/496
- mel FN (argmax) → nv×4 (conf 0.70–0.91), bkl×4 (conf 0.50–1.00), akiec×1 (0.96), bcc×1 (0.99) — 高確信度誤分類が支配的
- Binary FN内訳: mel×2 (P(mal) 0.011, 0.108), bcc×1 (0.018), akiec×3 (0.032–0.098), mel×1 (0.136) — 全件P(mal)<0.14、良性クラスタに埋没

### 定性的考察

**クラスタ構造**: v3a は良悪性の大枠分離が明確（左半分=悪性、右半分=NV）。v8 は BCC/AKIEC が密なクラスタを形成しv3aより分離が鮮明（PanDerm backbone が表皮系腫瘍の視覚パターンを事前学習段階で捉えている示唆）。v10 は6クラスが空間全体に入り混じり良悪性分離すら不明瞭（臨床写真の情報量の限界）。

**mel は全モデルでクラスタを形成しない**: v3a で2群、v8 で3群以上に分裂。メラノーマの視覚的サブタイプ（色素性/無色素性/結節型/表在拡大型）が特徴空間上で異なる場所に配置されるためで、モデルの問題ではなく単一ラベルが視覚的に異質な集団を包含していることの反映。

**二値閾値後のFN構造**: v3a FN 7件・v8 FN 2件とも良性クラスタ内部に埋もれ、P(mal)が閾値を大きく下回る（v3a: 0.01–0.14、v8: 0.05–0.07）。閾値をさらに下げても救済困難な構造的FN。v8の優位はargmaxでの確信度の低さ（フラットな確率分布）→ P(mal)集約での救済率の高さに起因。

**pretraining効果**: ImageNet FT (v3a) は7クラスargmax精度に最適化、PanDerm LP (v8) はBCC/AKIECをbackboneレベルで分離するがmelの統合はできない（self-supervisedでmelラベル未使用のため）。二値安全網の品質はpretraining由来の表現品質で決まる。

**post-hocパイプライン設計根拠**: t-SNE上のFN位置パターンから、閾値調整で救済可能な症例（P(mal)が閾値付近）と構造的に救済不可能な症例（良性クラスタに埋没、P(mal)≈0）を区別できた。Temperature Scaling → 二値集約 → 臨床閾値というパイプラインの設計判断を定性的に裏付ける。

### 限界

- t-SNEは局所構造のみ保存。クラスタ間距離に定量的意味はない
- v3a/v8/v10のt-SNEは座標系が異なり、位置の直接比較は不可能（分離パターンの定性比較のみ）
- N=552 (mel 23件) / N=475 のため、クラスタ形状はサンプリングの偶然に左右されうる
- 定量的埋め込み品質評価（silhouette score等）は未実施

## 実験設計の原則

訓練・post-hoc・calibrationは役割が異なり、改善対象も異なる。

| 層          | 手法                 | 改善対象                      | タイミング             | Notebook  |
| ----------- | -------------------- | ----------------------------- | ---------------------- | --------- |
| 訓練        | OS、CW、FL           | **ROC-AUC**（判別能力）       | モデルの重みを変える   | v3a / v3b |
| post-hoc    | 閾値最適化、二値集約 | **Recall / F1**（判定ルール） | モデルの重みは変えない | v5        |
| calibration | Temperature Scaling  | **ECE**（確率の信頼性）       | モデルの重みは変えない | v4        |

**post-hocはcalibration後に実施する。**
理由: malignant 3クラスのsum集約（`p_mal = p_mel + p_bcc + p_akiec`）はTSの単調変換性を破るため、TS→post-hocの順序が必須。

**3分割における評価の役割分担:**

- val: TS最適化・閾値最適化のベース（ハイパーパラメータチューニング用）
- test: 最終報告専用（閾値最適化には使用しない）

| 臨床閾値 Precision (test)      | 0.462                             | 0.646                                     |
| ECE（TS後、val）               | 0.0289                            | 0.0505                                    |

**DenseNet v3a が正当に主張できること**

- HAM10000 はダーモスコピーのデファクトスタンダードベンチマーク。Haenssle 2018 等の先行研究と比較可能な土台があり、0.9783/0.9615 は「HAMベンチマーク上の性能」として正当性がある
- 臨床写真への汎化は主張できない（DDI AUC 0.583 はchance levelに近い）

**PanDerm+Fitz v10 が正当に主張できること**

- Fitz test は patient leakage の懸念（Fitz17k は patient ID 非公開）があり 0.9089 は過大評価の可能性あり。参考値・caveat付きで示す
- **v10 の本命の主張は DDI AUC 0.675**（生検確認済み・独立データセット・臨床写真）
- DenseNet との比較: DDI overall +9.2pt、FST V-VI +5.8pt

**共通の誠実な主張**
どちらも「完璧な独立外部検証はない」。DenseNet はモダリティ内精度、v10 は外部バイアス評価でそれぞれ異なる主張をしており、用途が異なるモデルとして位置づける。

### 先行研究との DDI 性能比較

DDI データセット（Daneshjou et al., _Science Advances_ 2022）はバイオプシー確認済み 656 枚の独立外部評価セットであり、皮膚科 AI のバイアス評価標準として広く用いられている。以下は同データセット上での各モデルの比較である。

| モデル                    | 開発元                   | 自社テスト AUC | DDI Binary AUC（95% CI） | 備考                    |
| ------------------------- | ------------------------ | -------------- | ------------------------ | ----------------------- |
| DeepDerm                  | Google Health / Stanford | 0.88           | **0.56**（0.51–0.61）    | 臨床写真＋DS混合訓練    |
| ModelDerm                 | 韓国・서울大             | 0.93–0.94      | **0.65**（0.61–0.70）    | 臨床写真特化            |
| HAM10000 ベースモデル     | 同論文 baseline          | 0.92           | **0.67**（0.62–0.71）    | DS専用訓練              |
| **v10（本プロジェクト）** | PanDerm + Fitz17k        | 0.909\*        | **0.675**                | \*leakage懸念あり参考値 |
| v7（本プロジェクト）      | DenseNet + HAM           | 0.9615         | 0.583                    | DS訓練→臨床写真評価     |

**読み方：**

v10 の DDI AUC 0.675 は、Daneshjou et al. 2022 で報告された 3 モデルの中で最も高い HAM10000 baseline（0.67、95% CI 0.62–0.71）とほぼ同水準に位置する。CIの重複を考慮すると、この差は統計的に有意ではない可能性が高い。
v10 を「デプロイ可能」と判断する根拠としてこの数値を用いることはしない。理由は、独立外部評価が DDI 単独（n=656）であること、Fitz17k と DDI の分布差が残ること、DDI 臨床閾値での Precision が 0.315 と実運用上の問題を示していること、の 3 点による。
v10 の実験的価値は別の文脈にある。PanDerm frozen backbone に Fitz17k で head を追加訓練することで、DS専用 DenseNet（DDI 0.583）と比較して臨床写真モデル（DDI 0.675）の差が定量的に示せた。これは「モダリティと訓練分布が性能に与える影響」を測定した実証として位置づける。

**フィールド横断の観察：**

「自社テスト → 独立外部評価」の落差はすべてのシステムで発生している。MelaFind は FDA 認可後に特異度の臨床的無意味性で撤退、DermaSensor は認可されたが特異度 20.7% という問題を引き継いでいる。DermAssist は CE マーク取得後も DDI での感度が皮膚科医を下回る。「完全な独立外部検証を持つデプロイ済みシステムは現存しない」というのはフィールド全体の事実であり、DermLens 固有の問題ではない。

---

### フィールド全体のパターン

DDI 評価では、既存モデルは自社テスト結果と比較して ROC-AUC が 27–36% 低下した。 MIDAS 前向き研究でも ModelDerm は自社後ろ向き検証の AUC 0.86 から 0.68 へ低下した。

これは個別モデルの問題ではなく分野全体のパターンであり、「独立外部評価で自社値を維持できたシステムは現時点で存在しない」という段階にある。DermLens も例外ではない。
